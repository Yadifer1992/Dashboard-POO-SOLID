#  Dashboard POO SOLID

Este proyecto es una adaptación del repositorio original de Programación Orientada a Objetos, diseñado para organizar tareas y proyectos por unidades utilizando principios de diseño SOLID en Python.

## Estructura del Proyecto

El programa permite navegar por unidades (Unidad 1, Unidad 2, etc.) y acceder a los scripts organizados dentro de cada una.
* Adaptaciones Realizadas

    Se personalizó el menú principal para detectar dinámicamente las unidades.

    Se ordenan los scripts dentro de subcarpetas por unidad.

    Se aplicaron principios SOLID para estructurar mejor el código.

    Se mejoró la organización para facilitar la gestión de tareas del curso.
